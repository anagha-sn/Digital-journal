Project Overview

My Sunny Day Journal is an interactive, mood-based digital journaling web application designed to make self-reflection engaging, creative, and visually expressive.

Unlike traditional text-only journals, this platform allows users to:
Select their mood
Express emotions visually using a drawing canvas
Experience playful animations
Save their daily reflections in a fun and comforting environment
The goal of this project is to promote emotional awareness and creativity through a simple, user-friendly web interface.
